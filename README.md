


For CDN(Contents Deliver Network) supported by [jsdelivr](https://www.jsdelivr.com/?docs=gh) to access files inside this repo in elsewhere.

Format to access any file in any repository (use either one):
- https://cdn.jsdelivr.net/gh/github_username/github_repo/file_name
- https://cdn.jsdelivr.net/gh/github_username/github_repo@commit_hash/file_name

Examples:
- https://cdn.jsdelivr.net/gh/chewzzz1014/file/train.csv.zip 
- https://cdn.jsdelivr.net/gh/chewzzz1014/file@b1ab3b5ff3755b0fca0fbee3ad20ee0cf73bf09a/UML.drawio.png
